angular.json
Contains Angular project configuration like build options, assets, styles, scripts and environments.

tsconfig.json
Base TypeScript configuration used by the project.

tsconfig.app.json
TypeScript configuration specific to the Angular application.

package.json
Contains project dependencies, scripts and metadata.

src/main.ts
Application entry point that bootstraps the root Angular application.

src/app/app.config.ts
Defines providers and application configuration for standalone Angular applications.

src/app/app.component.ts
Root component of the application.

src/index.html
Main HTML page where the Angular application is loaded.

maximumWarning
Shows a warning if the application bundle exceeds the specified size.

maximumError
Fails the build if the bundle exceeds the specified limit.
